### 1.3.6 Enterprise Conversational

<p align="justify"> The Enterprise Conversational scenario is based on D1.1 [2], User Scenario 4 – Smart Enterprises -Contextual Enriched Communication in Smart Enterprises. It will be supported by an application
delivering enriched human to human communication in an enterprise context. This application allows collaborating with company’s co-workers, benefiting for enriched presence status in the application
directory but also with external partners or new and existing clients. It allows scheduling seamless multiparty conferences. It also allows using enhanced quality communication that would be required
for important communications/meetings. In this scenario, business identity is mainly used and usage of business brokers may be used for setting up contextual relationships between companies.</p>
